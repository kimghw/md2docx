# 작업 지침

## 자료 검색 순서

1. **references 폴더 우선 검색**
   - 작업에 필요한 정보는 먼저 [references/](references/) 폴더 안의 자료를 확인한다.
   - 관련 문서가 있으면 해당 내용을 기반으로 답변/작업한다.

2. **필요 시 웹 검색**
   - references 에 관련 자료가 없거나, 내용이 부족한 경우 웹 검색으로 최신 정보를 보완한다.
   - 공식 문서, 신뢰할 수 있는 출처를 우선한다.

3. **references 갱신**
   - 웹 검색 등으로 새로 알게 된 내용이나 변경사항이 있으면 [references/](references/) 내 해당 파일을 수정/추가한다.
   - 기존 문서에 오래된 정보가 있으면 최신 내용으로 업데이트한다.
   - 새로운 주제는 별도 `.md` 파일로 references 폴더에 추가한다.
