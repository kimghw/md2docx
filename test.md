# 제목 1 — 대분류

이 문서는 `md-docx-transform` 스킬이 Markdown의 주요 요소를 Word 스타일로 올바르게 변환하는지 확인하기 위한 테스트입니다. 첫 문단은 `First Paragraph` 스타일로 처리될 수 있습니다.

## 인용문

> 인용문은 `Block Text` 또는 `Quote` 스타일로 변환됩니다.
> 두 번째 줄도 같은 스타일이 이어집니다.

## 코드

인라인 코드: `pandoc --reference-doc=template.docx input.md -o output.docx`

코드 블록 (`Source Code` 스타일):

```python
def convert(md_file, template, output):
    """Markdown을 DOCX로 변환하고 tblLook을 보정."""
    subprocess.run(["pandoc", md_file,
                    f"--reference-doc={template}",
                    "-o", output])
    fix_tbllook(output)
```
